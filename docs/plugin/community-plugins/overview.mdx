# Обзор

## Плагины сообщества

Вы можете ознакомиться с плагинами Rspress, предоставленными сообществом, на странице [awesome-rstack - Rspress Plugins](https://github.com/rstackjs/awesome-rstack?tab=readme-ov-file#rspress-plugins).

Вы также можете найти больше плагинов Rspress на npm, выполнив поиск по ключевому слову [rspress-plugin](https://www.npmjs.com/search?q=rspress-plugin&ranking=popularity).

Вот некоторые плагины от сообщества:

- [rspress-plugin-file-tree](https://github.com/rstackjs/rspress-plugins/tree/main/packages/rspress-plugin-file-tree): плагин Rspress, добавляющий древовидное представление для отображения структуры файлов.
- [rspress-plugin-gh-pages](https://github.com/rstackjs/rspress-plugins/tree/main/packages/rspress-plugin-gh-pages): плагин Rspress для добавления поддержки автоматического развертывания на GitHub Pages.
- [rspress-plugin-translate](https://github.com/byteHulk/rspress-plugin-translate): Плагин, который интегрирует LLM для перевода документов.
- [rspress-plugin-font-open-sans](https://github.com/rstackjs/rspress-plugin-font-open-sans): Использование шрифта Open Sans в качестве шрифта по умолчанию на сайте Rspress.
- [rspress-plugin-align-image](https://github.com/rstackjs/rspress-plugins/tree/main/packages/rspress-plugin-align-image): Плагин Rspress для выравнивания изображений в разметке Markdown.
- [rspress-plugin-directives](https://github.com/rstackjs/rspress-plugins/tree/main/packages/rspress-plugin-directives): Плагин Rspress для поддержки пользовательских директив.
- [rspress-plugin-google-analytics](https://github.com/rstackjs/rspress-plugins/tree/main/packages/rspress-plugin-google-analytics): Плагин Rspress для интеграции с Google Analytics.
- [rspress-plugin-vercel-analytics](https://github.com/rstackjs/rspress-plugins/tree/main/packages/rspress-plugin-vercel-analytics): Плагин Rspress для интеграции с Vercel Analytics.
- [rspress-plugin-katex](https://github.com/rstackjs/rspress-plugins/tree/main/packages/rspress-plugin-katex): Плагин Rspress для поддержки рендеринга математических уравнений с использованием [KaTeX](https://katex.org/).
- [rspress-plugin-live2d](https://github.com/rstackjs/rspress-plugins/tree/main/packages/rspress-plugin-live2d): Плагин Rspress для поддержки Live2D, использующий [on-my-live2d](https://oml2d.com/).
- [rspress-plugin-mermaid](https://github.com/rstackjs/rspress-plugins/tree/main/packages/rspress-plugin-mermaid): Плагин Rspress для рендеринга диаграмм [Mermaid](https://mermaid.js.org/#/) в файлах Markdown.
- [rspress-plugin-reading-time](https://github.com/rstackjs/rspress-plugins/tree/main/packages/rspress-plugin-reading-time): Плагин Rspress для отображения времени чтения для страниц документации.
- [rspress-plugin-supersub](https://github.com/rstackjs/rspress-plugins/tree/main/packages/rspress-plugin-supersub): Плагин Rspress для поддержки верхнего индекса (`<super></super>`) и нижнего индекса (`<sub></sub>`).
- [rspress-plugin-toc](https://github.com/rstackjs/rspress-plugins/tree/main/packages/rspress-plugin-toc): Плагин Rspress, который вставляет оглавление на страницу.
- [rspress-plugin-auto-meta](https://github.com/smileluck/rspress-plugin-auto-meta.git): Плагин для Rspress, который автоматически генерирует метаданные навигации для вашей документации.
- [rspress-plugin-map](https://github.com/buyfakett/rspress-plugin-map.git): вставка интерактивных карт в Rspress, вдохновлено [hexo-tag-map](https://github.com/kuole-o/hexo-tag-map.git).
- [rspress-plugin-auto-sidebar](https://github.com/buyfakett/rspress-plugin-auto-sidebar.git): автоматическая генерация боковой панели на основе конфигурации навигационной панели.
- [rspress-plugin-giscus](https://github.com/buyfakett/rspress-plugin-giscus.git): интеграция [giscus](https://github.com/giscus/giscus) в Rspress, системы комментариев, работающей на базе GitHub Discussions.
- [rspress-plugin-blog-list](https://github.com/buyfakett/rspress-plugin-blog-list.git): интеграция списка блогов в Rspress.

## Вклад в плагины сообщества

Вклады сообщества в плагины Rspress приветствуются! Имена npm-пакетов плагинов должны следовать соглашению с префиксом `rspress-plugin-`.

Вы можете внести свой вклад следующими способами:

- Разработайте и опубликуйте плагин в собственном репозитории, затем отправьте PR, чтобы добавить его в список плагинов сообщества на этой странице.
- Внесите вклад напрямую в репозиторий [rspress-plugins](https://github.com/rstackjs/rspress-plugins), который уже содержит несколько плагинов сообщества, которые могут служить примером.
