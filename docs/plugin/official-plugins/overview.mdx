# Обзор

## Официальные плагины

Официальные плагины включают:

- [@rspress/plugin-algolia](./algolia): Заменяет встроенный поиск Rspress на [Algolia](https://www.algolia.com/), основанный на [DocSearch](https://docsearch.algolia.com).
- [@rspress/plugin-api-docgen](./api-docgen): Интегрирует [react-docgen-typescript](https://github.com/styleguidist/react-docgen-typescript) и [документацию](https://github.com/documentationjs/documentation) для автоматической генерации описаний API-документации.
- [@rspress/plugin-client-redirects](./client-redirects): Поддерживает клиентские перенаправления.
- [@rspress/plugin-llms](./llms): Генерирует файлы [llms.txt](https://llmstxt.org/) для сайта Rspress, позволяя большим языковым моделям лучше понимать ваш сайт документации.
- [@rspress/plugin-playground](./playground): Предоставляет интерактивную площадку для просмотра блоков кода в Markdown/MDX файлах в реальном времени.
- [@rspress/plugin-preview](./preview): Поддерживает предварительный просмотр блоков кода в Markdown/MDX.
- [@rspress/plugin-rss](./rss): Генерирует RSS-файлы с помощью [feed](https://github.com/jpmonette/feed) для сайта документации.
- [@rspress/plugin-sitemap](./sitemap): Автоматически генерирует [sitemap](https://developers.google.com/search/docs/crawling-indexing/sitemaps/overview) для SEO, облегчая индексацию поисковыми системами.
- [@rspress/plugin-twoslash](./twoslash): Интегрирует [Twoslash](https://github.com/twoslashes/twoslash) для автоматической генерации богатых блоков кода с информацией о типах.
- [@rspress/plugin-typedoc](./typedoc): Интегрирует [TypeDoc](https://github.com/TypeStrong/typedoc) для автоматической генерации API-документации TS-модулей.
