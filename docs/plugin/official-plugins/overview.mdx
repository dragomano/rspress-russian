# Обзор

## Официальные плагины

Официальные плагины включают:

- [@rspress/plugin-algolia](./algolia): Заменяет встроенный поиск Rspress на [Algolia](https://www.algolia.com/), основанный на [DocSearch](https://docsearch.algolia.com/).
- [@rspress/plugin-api-docgen](./api-docgen): Интегрирует пакеты [react-docgen-typescript](https://github.com/styleguidist/react-docgen-typescript) и [documentation](https://github.com/documentationjs/documentation) для автоматической генерации API-справки.
- [@rspress/plugin-client-redirects](./client-redirects): Добавляет клиентские редиректы.
- [@rspress/plugin-llms](./llms): Генерирует файлы, связанные с [llms.txt](https://llmstxt.org/), для сайта Rspress, чтобы LLM лучше понимали документацию.
- [@rspress/plugin-playground](./playground): Предоставляет интерактивную площадку для блоков Markdown/MDX.
- [@rspress/plugin-preview](./preview): Предпросмотр блоков кода Markdown/MDX.
- [@rspress/plugin-rss](./rss): Генерирует RSS-фиды для документационного сайта с использованием [feed](https://github.com/jpmonette/feed).
- [@rspress/plugin-sitemap](./sitemap): Автоматически генерирует sitemap для SEO и индексации поисковыми системами.
- [@rspress/plugin-twoslash](./twoslash): Интегрирует [Twoslash](https://github.com/twoslashes/twoslash) для автоматической генерации расширенных блоков кода с типами.
- [@rspress/plugin-typedoc](./typedoc): Интегрирует [TypeDoc](https://github.com/TypeStrong/typedoc) для автоматической генерации API-документации для TypeScript-модулей.
