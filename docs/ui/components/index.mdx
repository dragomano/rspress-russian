title: Компоненты документации
---

Rspress предоставляет часто используемые компоненты для написания контента документации, которые можно напрямую импортировать в MDX-файлах. Например:

```mdx title="docs/index.mdx"
import { PackageManagerTabs } from '@rspress/core/theme';

<PackageManagerTabs command="create rspress@latest" />
```
