# Steps

`Steps` отображает Markdown-контент в виде пошаговых инструкций.

## Использование

import { Steps } from '@rspress/core/theme';

<Steps>

### Шаг 1

Текст для шага 1.

### Шаг 2

> Текст для шага 2.

</Steps>

```mdx title="index.mdx"
import { Steps } from '@rspress/core/theme';

<Steps>

### Шаг 1

Текст для шага 1.

### Шаг 2

> Текст для шага 2.

</Steps>
```

Внутри `<Steps>` пишите заголовки; каждый заголовок начинает новый шаг. Markdown под заголовком становится содержимым этого шага.
