# Prompt

`Prompt` выделяет многоразовые инструкции, которые читатели могут скопировать и вставить непосредственно в свои AI-инструменты.

## Использование

Импортируйте `Prompt` в ваш MDX-файл перед его использованием:

```mdx title="index.mdx"
import { Prompt } from '@rspress/core/theme';

<Prompt
  title="Запустите проект Rspress одной командой"
  description="Скопируйте этот промпт и отправьте его своему AI-агенту. Он автоматически создаст для вас новый сайт Rspress."
  prompt={`Создай проект Rspress, следуя официальному руководству по быстрому старту по адресу
https://rspress.rs/guide/start/getting-started.md.

1. Создай проект с помощью \`create rspress@latest\`.
2. Установи зависимости.
3. Запусти сервер разработки.
4. Убедись, что сайт отображается корректно.`}
/>
```

Содержимое промпта может быть свёрнуто. Нажмите в любом месте карточки, чтобы скопировать полный текст промпта.

## Пользовательский контент

Установите `custom`, чтобы оставить только верхний колонтитул и внешнюю рамку, затем отобразите внутреннее содержимое с помощью дочерних элементов MDX. Пользовательские промпты не включают поведение копирования или сворачивания.

```mdx title="index.mdx"
<Prompt custom>

**привет**, мир

</Prompt>
```

<Prompt custom>

**привет**, мир

</Prompt>

:::tip
Если содержимое промпта длинное, вы можете поместить его в отдельный файл и импортировать через `?raw`:

```mdx title="index.mdx"
import { Prompt } from '@rspress/core/theme';
import promptText from './_prompt.txt?raw';

<Prompt
  title="Запустите проект Rspress одной командой"
  description="Скопируйте этот промпт и отправьте его своему AI-агенту."
  prompt={promptText}
/>
```

:::

## Использование с кастомными темами

Вы можете извлечь компонент `Prompt`, чтобы настроить его стили и поведение:

```bash
npx rspress eject Prompt
```

Это копирует компонент в `theme/components/Prompt`. После настройки повторно экспортируйте его в вашем `theme/index.tsx`:

```tsx title="theme/index.tsx"
export { Prompt } from './components/Prompt';
export * from '@rspress/core/theme-original';
```

## Пропсы

```ts
interface PromptProps extends React.HTMLAttributes<HTMLDivElement> {
  /**
   * Отрисовка пользовательского MDX-контента без поведения копирования или сворачивания.
   * @default false
   */
  custom?: boolean;
  /**
   * Встроенное описание, отображаемое в заголовке промпта.
   */
  description?: string;
  /**
   * Метка заголовка для блока промпта.
   * @default 'Agent Prompt'
   */
  title?: string;
  /**
   * Верхний колонтитул, отображаемый над заголовком.
   * @default 'For your Agent'
   */
  eyebrow?: string;
  /**
   * Управляет начальным свёрнутым состоянием.
   * @default true
   */
  defaultCollapsed?: boolean;
  /**
   * Текст промпта для отображения и копирования.
   */
  prompt?: string;
}
```
