# CSS-переменные

Rspress предоставляет часто используемые CSS-переменные. Переопределение переменных проще и удобнее в поддержке, чем замена встроенных React-компонентов. См. раздел [CSS-переменные](/guide/basic/custom-theme#css-variables), чтобы узнать, как переопределять эти стили.

:::tip Совет

Вы можете использовать функцию «Copy Markdown» на этой странице, чтобы ваш AI-агент помог вам изменить CSS-переменные.

:::

Ниже перечислены некоторые CSS-переменные, предоставляемые Rspress, и их значения по умолчанию:
