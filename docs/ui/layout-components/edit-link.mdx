# EditLink

`EditLink` отображает ссылку «Отредактировать страницу», которая ведёт на страницу редактирования исходного файла документа (например, на GitHub).

## Использование

Этот компонент автоматически отображается как в [DocFooter](/ui/layout-components/doc-footer), так и в панели оглавления, и не требует ручного использования.

```tsx preview
import { EditLink as BasicEditLink } from '@rspress/core/theme-original';

export default function EditLink() {
  return <BasicEditLink />;
}
```

## Свойства

| Свойство     | Тип       | Значение по умолчанию | Описание                                                                                   |
| ------------ | --------- | --------------------- | ------------------------------------------------------------------------------------------ |
| `isOutline`  | `boolean` | `false`               | Когда `true`, отображает компактный стиль с иконкой редактирования для использования в панели оглавления. Когда `false`, отображает стандартный стиль текстовой ссылки для футера. |

## Связанная конфигурация

Настройка `editLink` в `rspress.config.ts`:

```ts title="rspress.config.ts"
import { defineConfig } from '@rspress/core';

export default defineConfig({
  themeConfig: {
    editLink: {
      docRepoBaseUrl:
        'https://github.com/web-infra-dev/rspress/tree/main/website/docs',
      text: '📝 Редактировать эту страницу на GitHub',
    },
  },
});
```

- `docRepoBaseUrl` — Базовый URL-адрес репозитория документации
- `text` — Текст, отображаемый для ссылки

Если `editLink` не настроен, компонент ничего не отобразит.
