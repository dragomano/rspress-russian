# PrevNextPage

`PrevNextPage` отображает ссылки навигации по предыдущей/следующей странице в нижней части страницы.

## Использование

```tsx title="index.mdx"
import { PrevNextPage } from '@rspress/core/theme';

<PrevNextPage />;
```

Этот компонент не принимает никаких пропсов. Он автоматически рассчитывает ссылки на предыдущую/следующую страницу на основе порядка в боковой панели.

## Как это работает

Компонент внутренне использует хук `usePrevNextPage` для получения информации о предыдущей и следующей странице:

```ts
import { usePrevNextPage } from '@rspress/core/theme';

const { prevPage, nextPage } = usePrevNextPage();
// prevPage: { text: string; link: string } | null
// nextPage: { text: string; link: string } | null
```

- Страницы «предыдущая/следующая» автоматически рассчитываются на основе позиции текущей страницы в боковой панели
- Если текущая страница первая, `prevPage` будет `null`
- Если текущая страница последняя, `nextPage` будет `null`
