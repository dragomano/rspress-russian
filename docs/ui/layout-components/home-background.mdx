# HomeBackground

Этот компонент является частью [главной страницы](/guide/basic/home-page). `HomeBackground` отображает фоновый эффект на главной странице и автоматически устанавливает прозрачный стиль для панели навигации.

## Использование

Вы можете eject'ить этот компонент для его кастомизации.

```mdx title="index.mdx"
import { HomeBackground } from '@rspress/core/theme';

<HomeBackground className="my-custom-bg" />
```

## Пропсы

Этот компонент наследует все свойства `React.HTMLAttributes<HTMLDivElement>`.

### className

- **Тип:** `string`

Добавьте пользовательские имена CSS-классов для кастомизации стиля фона.

## Как это работает

- Компонент автоматически переключает панель навигации в прозрачный режим
- Вы можете кастомизировать класс `.rp-home-background` через CSS, чтобы настроить фоновый эффект
