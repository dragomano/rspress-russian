# LastUpdated

`LastUpdated` отображает время последнего обновления страницы и, при необходимости, автора последнего изменения. Обычно используется вместе с параметром `lastUpdated` в конфигурации темы.

## Использование

```tsx preview
import { LastUpdated as BasicLastUpdated } from '@rspress/core/theme-original';

export default function LastUpdated() {
  return <BasicLastUpdated />;
}
```

Этот компонент не принимает никаких пропсов. Он автоматически читает время последнего обновления из метаданных страницы.

## Связанная конфигурация

Включите `lastUpdated` в `rspress.config.ts`:

```ts title="rspress.config.ts"
import { defineConfig } from '@rspress/core';

export default defineConfig({
  themeConfig: {
    lastUpdated: true,
  },
});
```

Когда `lastUpdated` включён, компонент автоматически читает и форматирует метаданные страницы для отображения.

### Отображение автора

Настройте `lastUpdated.author`, чтобы отображать автора последнего коммита:

```ts title="rspress.config.ts"
export default defineConfig({
  themeConfig: {
    lastUpdated: {
      // Показывать имя автора коммита:
      // author: true,

      // Или использовать собственный формат отображения:
      author: ({ name, email }) => `${name} <${email}>`,
    },
  },
});
```

Функция обратного вызова получает объект `{ name, email, filePath }` (информацию об авторе коммита из `git log` и путь к исходному файлу) и должна вернуть строку, которая будет отображаться.

Чтобы кастомизировать метку или форматирование, выполните `rspress eject LastUpdated` и отредактируйте файл `theme/components/LastUpdated/index.tsx`.
