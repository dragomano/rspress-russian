# NoSSR

`NoSSR` пропускает серверный рендеринг для своих дочерних элементов.

## Использование

```mdx title="index.mdx"
import { NoSSR } from '@rspress/core/runtime';

<NoSSR>
  <ThirdPartyWidget />
</NoSSR>
```

Используйте его, когда компонент зависит от API, доступных только в браузере, или когда нужно избежать серверного рендеринга (SSR).
