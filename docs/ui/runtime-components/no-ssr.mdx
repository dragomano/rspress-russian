# NoSSR

:::danger Устарело

`NoSSR` устарел. Используйте [`BrowserOnly`](./browser-only) вместо него.

:::

`NoSSR` использовался для пропуска серверного рендеринга своего поддерева.

## Использование

```mdx title="index.mdx"
import { NoSSR } from '@rspress/core/runtime';

<NoSSR>
  <ThirdPartyWidget />
</NoSSR>
```

## См. также

- [`BrowserOnly`](./browser-only) — рендеринг контента только в браузере после гидратации.
