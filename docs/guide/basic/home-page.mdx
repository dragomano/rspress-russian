# Главная страница

Rspress предоставляет готовый макет главной страницы. Установите `pageType: home` в метаданных, чтобы быстро создать рабочую главную страницу сайта.

Определите содержимое главной страницы через метаданные. Полные типы см. в разделе [Конфигурация метаданных](/api/config/config-frontmatter.html#hero). Ниже приведён простой пример:

```yaml title="docs/index.mdx"
---
pageType: home
title: Rspress
titleSuffix: 'Статический генератор сайтов на базе Rsbuild'

hero:
  name: Rspress
  text: Решение для документации
  tagline: Современный технологический стек для разработки документации
  actions:
    - theme: brand
      text: Введение
      link: /guide/introduction
    - theme: alt
      text: Быстрый старт
      link: /guide/getting-started

features:
  - title: 'Поддержка MDX'
    details: MDX — мощный способ написания контента. Вы можете использовать React-компоненты прямо в Markdown.
    icon: 📦
  - title: 'Богатый функционал'
    details: Поддержка i18n, полнотекстового поиска и многого другого из коробки.
    icon: 🎨
  - title: 'Гибкая настройка'
    details: Можно кастомизировать внешний вид темы и процесс сборки.
    icon: 🚀
---
```

## Компоненты домашней страницы

Домашняя страница Rspress состоит из следующих компонентов, которые вы можете настраивать через [Кастомную тему - Повторный экспорт ESM](/guide/basic/custom-theme#reexport):

- [HomeHero](/ui/layout-components/home-hero)
- [HomeFeature](/ui/layout-components/home-feature)
- [HomeFooter](/ui/layout-components/home-footer)
- [HomeBackground](/ui/layout-components/home-background)

:::tip

Вы также можете полностью настроить главную страницу, переопределив компонент [`HomeLayout`](/ui/layout-components/home-layout). В этом случае настройки, связанные с главной страницей, а также поля в метаданных перестанут применяться.

:::
