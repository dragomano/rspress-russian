# Расширение сборки

## Rsbuild

Rspress собирает документацию с помощью [Rsbuild](https://github.com/web-infra-dev/rsbuild).

### Настройка Rsbuild

Rsbuild предоставляет широкий набор параметров сборки. Настроить их можно с помощью [builderConfig](/api/config/config-build.html#builderconfig). Например, чтобы изменить каталог вывода на `doc_dist`:

```ts title="rspress.config.ts"
import { defineConfig } from '@rspress/core';

export default defineConfig({
  builderConfig: {
    output: {
      distPath: {
        root: 'doc_dist',
      },
    },
  },
});
```

Rspress также предоставляет [builderConfig.plugins](/api/config/config-build#builderconfigplugins) для регистрации плагинов Rsbuild. Используйте экосистему плагинов Rsbuild, чтобы расширить поведение сборки.

Например, можно добавить Google Analytics с помощью плагина [rsbuild-plugin-google-analytics](https://github.com/rstackjs/rsbuild-plugin-google-analytics):

```ts title="rspress.config.ts"
import { defineConfig } from '@rspress/core';
import { pluginGoogleAnalytics } from 'rsbuild-plugin-google-analytics';

export default defineConfig({
  builderConfig: {
    plugins: [
      pluginGoogleAnalytics({
        // замените на свой Google tag ID
        id: 'G-xxxxxxxxxx',
      }),
    ],
  },
});
```

:::tip
Подробнее см. в документации [Rsbuild — Config](https://rsbuild.rs/config/).
:::

### Настройка Rspack

Настроить Rspack можно через опцию [tools.rspack](https://rsbuild.rs/config/tools/rspack), которую предоставляет сам Rsbuild:

```ts title="rspress.config.ts"
import { defineConfig } from '@rspress/core';

export default defineConfig({
  builderConfig: {
    tools: {
      rspack(options) {
        // изменяем конфигурацию rspack
      },
    },
  },
});
```

## Компиляция MDX

Rspress компилирует MDX с помощью [unified](https://github.com/unifiedjs/unified). Добавить связанные плагины компиляции можно через конфигурацию `markdown`. Например:

```ts title="rspress.config.ts"
import { defineConfig } from '@rspress/core';

export default defineConfig({
  markdown: {
    remarkPlugins: [
      [
        require('remark-autolink-headings'),
        {
          behavior: 'wrap',
        },
      ],
    ],
    rehypePlugins: [require('rehype-slug')],
  },
});
```
